- Responsive one-page portfolio
- Dark and light themes
- Professional profile photo
- Résumé download

- Mobile navigation

- Accessibility improvements
- `robots.txt`, `sitemap.xml`, favicon and web manifest

## Publish on GitHub Pages

### 1. Create the repository

Create a **public** GitHub repository named exactly:

```text
Pratikshaa1216.github.io
```

### 2. Upload the files

Upload **all files and folders from this package** to the repository. Preserve the folder structure.

The root of the repository must contain:

```text
index.html
README.md
404.html
.nojekyll
site.webmanifest
robots.txt
sitemap.xml
assets/
```

### 3. Commit the files

Use a commit message such as:

```text
Launch personal portfolio
```

### 4. Enable GitHub Pages

Open:

```text
Repository → Settings → Pages
```

Under **Build and deployment** choose:

```text
Source: Deploy from a branch
Branch: main
Folder: / (root)
```

Click **Save**.

### 5. Open the website

After GitHub finishes publishing, the portfolio will be available at:

```text
https://pratikshaa1216.github.io/
```

Publishing can take several minutes after the first upload.

## Update the portfolio
